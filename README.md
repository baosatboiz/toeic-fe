# TOEIC FE

Frontend for an online TOEIC exam system (React + Vite).

## Features

- Login/Register flow
- Protected routes
- Exam session and timer handling
- Result page after exam submission

## Tech Stack

- React 19
- Vite
- React Router
- Bootstrap

## Setup

- `npm install`
- `npm run dev`

## Scripts

- `npm run dev` — start development server
- `npm run build` — build production bundle
- `npm run preview` — preview production build locally
